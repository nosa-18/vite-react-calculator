# Vite React Calculator

Jednostavna web aplikacija kalkulatora napravljena korišćenjem React, Vite i TypeScript tehnologija.

## Funkcionalnosti
- Sabiranje
- Oduzimanje
- Množenje
- Deljenje

## Tehnologije
- React
- Vite
- TypeScript

## Pokretanje projekta lokalno

1. Instalacija:
npm install

2. Pokretanje:
npm run dev
